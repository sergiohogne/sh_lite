# sh_lite
Reproductor de Audio Avanzado - Windows 10/11

SH Lite
Gracias por instalar SH Lite.
SH Lite es un reproductor de audio avanzado.
Permite gestionar listas de reproducción, carpetas aleatorias y programación horaria automática, con salida de audio configurable por dispositivo.
Características principales
•	Reproducción de archivos MP3, FLAC, OGG, M4A, M4B y WAV
•	Modos de reproducción: Simple, Lista y Loop
•	Carga de archivos individuales, carpetas completas y carpetas aleatorias con historial anti-repetición
•	Programación horaria automática con 96 franjas de 15 minutos
•	Medidores VU estéreo en tiempo real
•	Selección y bloqueo de dispositivo de salida de audio
•	Soporte bilingüe: Español e Inglés
•	Interfaz de 600×700 px, instancia única por sistema
Requisitos
•	Windows 10 Version 22H2 (64 bits)
Instalación
Ejecutar el instalador SH_Lite_Setup.exe y seguir los pasos en pantalla.
Uso básico
1.	Abrir SH Lite desde el acceso directo del escritorio o el menú inicio.

2.	Cargar archivos de audio con los botones Archivos, Carpetas o Aleatorio.

3.	Seleccionar el modo de reproducción y el dispositivo de salida.

4.	Presionar Play.
Para la programación horaria, hacer clic en el botón PG y configurar las franjas con los archivos de audio deseados.
