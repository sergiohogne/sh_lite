# SH Lite

Gracias por instalar SH Lite.

SH Lite es un reproductor de audio avanzado. Permite gestionar listas de reproducción, carpetas aleatorias y programación horaria automática, con salida de audio configurable por dispositivo.

## Características principales

- Reproducción de archivos MP3, FLAC, OGG, M4A, M4B y WAV
- Modos de reproducción: Simple, Lista y Loop
- Carga de archivos individuales, carpetas completas y carpetas aleatorias con historial anti-repetición
- Programación horaria automática con 96 franjas de 15 minutos
- Medidores VU estéreo en tiempo real
- Selección y bloqueo de dispositivo de salida de audio
- Soporte bilingüe: Español e Inglés
- Interfaz de 600×700 px, instancia única por sistema

## Requisitos

- Windows 10 Version 22H2 (64 bits)

## Instalación

Ejecutar el instalador `SH_Lite_Setup.exe` y seguir los pasos en pantalla.

## Uso básico

1. Abrir SH Lite desde el acceso directo del escritorio o el menú inicio.
2. Cargar archivos de audio con los botones **Archivos**, **Carpetas** o **Aleatorio**.
3. Seleccionar el modo de reproducción y el dispositivo de salida.
4. Presionar **Play**.

Para la programación horaria, hacer clic en el botón **PG** y configurar las franjas con los archivos de audio deseados.

## Documentación

https://sh.com.ar/sh_lite

# SH Lite

Thank you for installing SH Lite.

SH Lite is a professional audio player. It supports playlist management, random folder playback with anti-repeat history, and automatic time-based scheduling with configurable audio output devices.

## Main features

- Playback of MP3, FLAC, OGG, M4A, M4B and WAV files
- Playback modes: Single, List and Loop
- Load individual files, full folders, or random folders with anti-repeat history
- Automatic hourly scheduler with 96 slots of 15 minutes each
- Real-time stereo VU meters
- Audio output device selection and locking
- Bilingual interface: Spanish and English
- 600×700 px fixed window, single instance per system

## Requirements

- Windows 10 22H2 or later (64-bit)

## Installation

Run `SH_Lite_Setup.exe` and follow the on-screen instructions.

## Basic usage

1. Open SH Lite from the desktop shortcut or Start menu.
2. Load audio files using the **Files**, **Folders** or **Random** buttons.
3. Select playback mode and audio output device.
4. Press **Play**.

For time-based scheduling, click the **PG** button and assign audio files to the desired time slots.
